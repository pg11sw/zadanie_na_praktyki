Program jest zrobiony w Pythonie 3. Żeby go uruchomić, trzeba mieć bibliotekę requests.
Powinna ona być preinstalowana w środowisku Anaconda.
W innym wypadku trzeba ją zainstalować poleceniem:
pip install requests

Aby uruchomić program, należy użyć polecenia:
python zadanie.py
